# Numerical Examples of δx_min

Assuming \(C/\alpha = 1\) and \(\gamma = 1\), with \(k_B = 1.38 \times 10^{-23}\ \mathrm{J/K}\).

## Example 1
- T = 4 K  
- ε = 1×10⁻²² J  
- N = 1×10⁶  
- τ = 1×10⁻³ s  
- δx_min ≈ 3.94×10⁻¹³ (normalized units)

## Example 2
- T = 300 K  
- ε = 1×10⁻²¹ J  
- N = 1×10⁹  
- τ = 1×10⁻² s  
- δx_min ≈ 7.17×10⁻¹⁴ (normalized units)

## Example 3
- T = 1 K  
- ε = 1×10⁻²⁴ J  
- N = 1×10⁷  
- τ = 1 s  
- δx_min ≈ 1.22×10⁻¹⁵ (normalized units)

## Notes
- Values are illustrative to understand trends: reducing T and ε, increasing N and τ improves resolution.  
- In real setups, \(C/\alpha\) incorporates instrumental factors and physical limits.
