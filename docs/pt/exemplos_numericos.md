# Exemplos Numéricos de δx_min

Assumindo \(C/\alpha = 1\) e \(\gamma = 1\), com \(k_B = 1{,}38 \times 10^{-23}\ \mathrm{J/K}\).

## Exemplo 1
- T = 4 K  
- ε = 1×10⁻²² J  
- N = 1×10⁶  
- τ = 1×10⁻³ s  
- δx_min ≈ 3,94×10⁻¹³ (unidades normalizadas)

## Exemplo 2
- T = 300 K  
- ε = 1×10⁻²¹ J  
- N = 1×10⁹  
- τ = 1×10⁻² s  
- δx_min ≈ 7,17×10⁻¹⁴ (unidades normalizadas)

## Exemplo 3
- T = 1 K  
- ε = 1×10⁻²⁴ J  
- N = 1×10⁷  
- τ = 1 s  
- δx_min ≈ 1,22×10⁻¹⁵ (unidades normalizadas)

## Observações
- Valores são ilustrativos para entender tendências: reduzir T e ε, aumentar N e τ melhora a resolução.  
- Em setups reais, \(C/\alpha\) incorpora fatores instrumentais e limites físicos.
