# Guia de Submissão em Preprints (arXiv e OSF)

## Preparar o manuscrito
- Consolidar o artigo em docs/pt/preprint.md e docs/en/preprint.md.
- Gerar PDF a partir do Markdown (Pandoc ou editor que exporte para PDF).

## arXiv
- Criar conta em https://arxiv.org.  
- Escolher categoria (ex.: physics.gen-ph, quant-ph).  
- Submeter PDF, metadados (título, autores, resumo, palavras-chave).  
- Aguardar moderação e publicação.

## OSF
- Criar conta em https://osf.io.  
- Criar projeto e adicionar preprint.  
- Fazer upload do PDF e metadados.  
- Publicar para obter DOI OSF (opcional).

## Recomendações
- Incluir links para o repositório GitHub e o DOI do Zenodo.  
- Adicionar CITATION.cff no repositório e citar o DOI no README.  
- Manter consistência entre versões PT e EN.
