# Conteúdo Essencial
### Do Macro ao Micro: Princípio da Resolução por Energia Mínima

## Fórmula principal

\\[
\\delta x_{\\min} = \\frac{C}{\\alpha} \\cdot \\frac{\\sqrt{k_B T + \\gamma \\varepsilon}}{\\sqrt{N \\cdot \\tau}}
\\]

## Variáveis (resumo)
- δx_min: resolução espacial mínima
- α: acoplamento efetivo do probe
- C: constante geométrica/experimental
- k_B T: ruído térmico
- ε: energia por probe
- γ: distúrbio do probe
- N: número de probes coerentes
- τ: tempo de coerência/integração
